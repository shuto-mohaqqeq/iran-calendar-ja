# Iran Calendar (日本語・ペルシア語対応)

イランの祝日・宗教行事をまとめたカレンダーです。
Googleカレンダーに追加して利用できます。

---

## 概要

本プロジェクトは以下を目的としています：

* イランの公的祝日と宗教行事の整理
* 日本語で内容が理解できるカレンダー
* ペルシア語原文の保持
* Googleカレンダーでの実用性

---

## 特徴

* 日本語 + ペルシア語（فارسی）併記
* 双方向テキスト（BiDi）問題に配慮したレイアウト
* 金曜日（جمعه）を週休日として表示
* Googleカレンダーでの表示最適化済み
* 軽量でシンプルな構造

---

## 使い方

以下のURLをGoogleカレンダーに追加してください：

https://raw.githubusercontent.com/shuto-mohaqqeq/iran-calendar-ja/refs/heads/main/iran_calendar_full.ics

### 手順

1. Googleカレンダーを開く
2. 「他のカレンダー」→「＋」
3. 「URLで追加」
4. 上記URLを貼り付け

---

## 表示仕様

* **SUMMARY**：日本語（短く簡潔）
* **DESCRIPTION**：

  * 1行目：ペルシア語+日本語


## 対応内容

* ノウルーズ（نوروز）関連行事
* イスラーム暦に基づく宗教行事
* シーア派イマームの生誕・殉教日
* イランの国家祝日

---

## 注意事項

* イスラーム暦は観測により変動するため、日付が前後する可能性があります
* Googleカレンダーの仕様により反映に時間がかかる場合があります
* キャッシュの影響で更新が即時反映されないことがあります
